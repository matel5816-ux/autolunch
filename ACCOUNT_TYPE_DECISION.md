# 🤔 LINE 帳號類型選擇指南

**問題**：我應該用個人帳號還是商用帳號來創建 AutoLunch Bot？

---

## 📊 帳號類型對比

### 選項 1：個人 LINE 帳號 👤

| 項目 | 說明 |
|------|------|
| **申請難度** | ⭐ 簡單（5 分鐘） |
| **成本** | 免費 |
| **用途限制** | 可用於商業（但不夠正式） |
| **品牌形象** | ❌ 個人名義，看起來不正式 |
| **管理權限** | 只有你能管理 |
| **推薦場景** | 個人專案、試驗性項目 |
| **適合程度** | ⭐⭐（有限） |

**優點**：
- 申請快（馬上開始開發）
- 無成本

**缺點**：
- 看起來不正式（Bot 顯示你的個人名字）
- 不適合公司正式使用
- 無法設置公司品牌

---

### 選項 2：LINE Business ID（商用帳號） 🏢

| 項目 | 說明 |
|------|------|
| **申請難度** | ⭐⭐ 中等（需驗證） |
| **成本** | 免費（但後期可能需要付費方案） |
| **用途限制** | ✅ 完全適合商業使用 |
| **品牌形象** | ✅ 公司名義，正式專業 |
| **管理權限** | 多人管理、角色權限 |
| **推薦場景** | 公司項目、正式上線 |
| **適合程度** | ⭐⭐⭐⭐⭐（完美） |

**優點**：
- 正式、專業的品牌形象
- Bot 名稱顯示為 "群健科技 AutoLunch Bot"
- 多人管理（可邀請同事管理）
- 後續擴展業務時無需更換帳號
- 可獲得官方支持

**缺點**：
- 需要公司資訊驗證（營業執照等）
- 申請時間可能 2-3 天
- 未來如果要用付費功能，費用會增加

---

### 選項 3：LINE Official Account（官方帳號）📱

| 項目 | 說明 |
|------|------|
| **申請難度** | ⭐⭐⭐ 複雜 |
| **成本** | 可能需付費（約 $10-50/月） |
| **用途限制** | ✅✅ 最完整的功能 |
| **品牌形象** | ✅✅ 最專業的認證 |
| **管理權限** | ✅ 完整的業務管理工具 |
| **推薦場景** | 公開商業服務、大規模應用 |
| **適合程度** | ⭐⭐⭐（過度） |

**優點**：
- 最正式、最專業
- 有驗證徽章
- 完整的分析報告

**缺點**：
- 申請流程複雜
- 可能需付費
- 對內部工具來說過度設計

---

## 🎯 **建議方案**

### 我的建議：**選項 2 - LINE Business ID**✅

**理由**：

```

✅ 打算長期使用
```

3. ✅ **易於管理**：可邀請同事共同管理
4. ✅ **未來擴展**：如果後來要付費升級，無需更換帳號
5. ✅ **符合需求**：最符合「公司內部工具」的定位

## 📋 申請步驟對比

### 快速方案（個人帳號）

```
時間：5 分鐘
步驟：
1. 用個人 LINE 帳號登入 https://developers.line.biz/
2. 建立 Provider → Channel
3. 複製 Secret 和 Token
4. 開始開發

缺點：Bot 顯示你的個人名字，看起來不正式
```

### 推薦方案（商用帳號）

```
時間：10 分鐘（申請）+ 1-2 天（審核）
步驟：
1. 前往 https://business.line.biz/
2. 選擇「建立帳號」→ 企業帳號
3. 填入群健科技資訊
4. 上傳營業執照（如需要）
5. 等待審核（通常 24 小時內）
6. 審核通過後，建立 Messaging API Channel
7. 複製 Secret 和 Token
8. 開始開發

優點：正式、專業、長期適用
```
